Some HTML tags and their use

The ul tag
This tag creates a list where the order of items listed is not important.
The ol tag
This tag creates an ordered list (e.g. numbered list or alhabetical list)
The Table tag
This tag creates a table in HTML and it consits of other elements namely: tr, th and td tags
a href tag
This tag specifies a link's destination
(e.g. ""Google homepage"" will take you to google's homepage)
The Heading tag
This tag serves as an introductory section to a web page
